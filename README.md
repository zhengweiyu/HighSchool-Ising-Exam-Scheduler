# HighSchool-Ising-Exam-Scheduler
## 项目介绍
本仓库包含高中适配版伊辛模型期末考试排程代码，基于模拟退火算法实现，用于解决5门科目（语文、数学、英语、物理、化学）的期末考试时段安排问题，规避科目冲突，适配高中信息技术课跨学科教学实践。

## 核心功能
1.  构建高中科目冲突矩阵（默认适配数学-物理、数学-化学、英语-物理冲突）；
2.  简化伊辛模型映射：科目对应自旋状态，冲突约束编码为自旋相互作用；
3.  模拟退火算法求解最优排程，输出时段安排与冲突数；
4.  可视化能量变化曲线，直观展示算法收敛过程。

## 依赖环境
- Python 3.9+
- numpy 1.24.0+
- matplotlib 3.6.0+

## 运行步骤
1.  克隆仓库到本地：git clone https://github.com/zhengweiyu/HighSchool-Ising-Exam-Scheduler.git
2.  安装依赖库：pip install numpy matplotlib
3.  运行代码：python ising_exam_scheduler.py
4.  查看结果：终端输出最优排程，自动弹出能量变化曲线图表。

## 教学适配说明
1.  代码简化：删除复杂类结构，采用线性流程+中文注释，适配高中生编程基础；
2.  场景贴近：以高中期末考试排程为载体，降低跨学科知识理解门槛；
3.  可拓展性：支持修改冲突矩阵（增加科目/调整冲突关系），适配课堂拓展练习。
